# Quick Notes Privacy Policy

Last updated: July 13, 2026

Quick Notes is a lightweight browser extension for saving plain-text notes locally in your browser.

## Data We Handle

Quick Notes may handle the following data only when you use the extension:

- Notes you type into the extension popup
- Text you explicitly select on a web page and save using the “Add to Quick Notes” context menu
- Tags you create for organizing notes
- Note metadata, such as creation timestamps and expanded/collapsed state

Quick Notes does not collect passwords, cookies, authentication tokens, payment information, browsing history, or full page contents.

## How We Use Data

The data is used only to provide the extension’s note-taking features:

- Saving notes
- Displaying notes in the popup
- Organizing notes with tags
- Copying notes to your clipboard when requested
- Exporting notes as a local text file when requested
- Saving selected page text as a plain-text note when you choose “Add to Quick Notes”

## Local Storage Only

All notes, tags, and related metadata are stored locally in your browser using the Chrome Storage API.

Quick Notes does not send your notes, selected text, tags, or browsing data to any external server.

## Browser Permissions

Quick Notes uses the following permissions:

- `storage`: to save notes and tags locally in your browser
- `contextMenus`: to show the “Add to Quick Notes” option when text is selected
- `activeTab` and `scripting`: to read only the text you selected after you click the context menu item, so the extension can preserve plain-text line breaks from elements such as paragraphs, list items, divs, and line breaks

These permissions are used only for the extension’s visible note-taking features.

## Data Sharing

Quick Notes does not sell, rent, transfer, or share your data with third parties.

Quick Notes does not use your data for advertising, analytics, profiling, retargeting, or marketing.

## Data Retention and Deletion

Your notes and tags remain stored locally until you delete them, clear your browser extension data, or uninstall the extension.

You can delete notes and tags directly inside the extension.

## Security

Quick Notes does not transmit user data over the network. Because data is stored locally in your browser profile, access to that data depends on the security of your browser, device, and Chrome profile.

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any material change in data handling will be reflected in an updated version of this policy.
